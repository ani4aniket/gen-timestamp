=========

A small library that generates current timestamp.

## Installation

  `npm install gen-timestamp`

## Usage

    var genTimestamp = require("gen-timestamp");

    var formattedNum = genTimestamp();
  
  
  Output should be the current timestamp.


## Tests

  `npm test`
